Software Recommendations

As a fan of the open source philosophy, I try to use free and open source projects whenever possible. Here is a list of the programs I recommend using, based on category of software.

Web Browers

  • Brave

    • Brave is a free, semi-open-source internet browser based in Chromium. It is customizeable, user-friendly, and comes with a ton of useful features.

Text Editors

  • Neovim

    • Neovim is free, open-source, and an improved version of the already amazing text editor Vim. It has more features, less bloat, and is more customizeable.

  • Vim

    • Vim is free, open-source, and an improved version of Vi. Vim is slightly more bloated than Neovim and has less features, however its one advantage is that Lua can be used for configuration.

Operating Systems

  • Artix Linux

    • Artix Linux is my personal favorite Linux distro. It's based on Arch Linux (my previous favorite). It's customizeable, fast, lightweight, and not overly-complicated. It's advantage over Arch is that it doesn't use systemd (which is old and slow).

  • Arch Linux

    • Arch Linux is my second favorite Linux distro. It's considered a classic among Linux users since it's been around forever and has served as the base for many other Linux distros. Like Artix, it's customizeable, fast, lightweight, and not overly-complicated.

  • Pop!_OS

    • Pop!_OS is a beginner Linux distro based in Ubuntu, without the spyware. It brings all the advantages of using Linux while being just as easy to use as MacOS or Windows.

Shells

  • zsh

    • zsh is super fast, its autocomplete is faster than Bash's and functions like a drop-down menu, it follows POSIX standards more closely than Bash, and is super customizeable.

  • Bash

    • Bash is the most widely used shell in Linux and comes installed by default on many UNIX systems. It gets the job done, however with alternatives available, it isn't necessarily my first choice.
